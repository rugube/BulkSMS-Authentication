# PlatformAPI 

### Installation

#### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/waltermwanyisa/authentication.git PlatformApi

# go into app's directory
$ cd PlatformApi

# install app's dependencies
$ npm install
```

#### Usage

``` bash
# serve with hot reload at localhost:8080
node index.js

or 

nodemon index


```
